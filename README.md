# AI Investment Advisor
Una IA para asesoramiento financiero basado en IA.

## Instalación
1. Clona el repo: `git clone https://github.com/Wenraymati/ai-investment-advisor.git`
2. Instala dependencias: `npm install`
3. Configura `.env` con `DATABASE_URL`, `JWT_SECRET`, etc.
4. Despliega en Railway.

## Uso
Accede a https://smartproia.com/app.html.
